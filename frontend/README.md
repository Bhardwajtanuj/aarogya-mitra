# 💻 Aarogya Mitra - Frontend (React)

A modern, responsive e-healthcare dashboard built with **React 18** and **Tailwind CSS**.

## 🎨 Design Principles
- **Clean Aesthetic:** Healthcare-themed color palette with high readability.
- **Responsive Layout:** Mobile-first approach using Tailwind's grid and flexbox.
- **Micro-interactions:** Smooth transitions and loading states for a premium feel.
- **Type Safety:** Prop validation and schema-based form handling.

## 🚀 Features
- **Patient Dashboard:** Appointment booking, report history, and ML prediction interface.
- **Doctor Portal:** Consultation management and patient history access.
- **Admin Panel:** Real-time statistics and user management.
- **Auth Flow:** Seamless login/register experience with persist-auth.

## 🛠️ Tech Stack
- **Vite:** Next-gen frontend tooling.
- **Lucide React:** Iconography.
- **Axios:** API communication.
- **Zod:** Form validation logic.

## 🚦 Local Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`

> **Note**: For a full-system launch (Backend + Frontend + ML), use the **`start-app.bat`** orchestrator in the project root.
